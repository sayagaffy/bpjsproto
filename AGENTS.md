# Direktif Proyek: Prototype Otomasi Laporan Keuangan BPJS (Fase 1)

## 1. Fokus Utama (Frontend Prototype Only)
- Proyek ini HANYA berfokus pada pembuatan Prototype UI interaktif untuk kebutuhan demonstrasi klien.
- DILARANG keras menginisialisasi database (Postgres) atau membangun logika backend (Node.js/Hono) pada tahap ini.
- Gunakan DUMMY DATA (file JSON statis yang disimpan di frontend) untuk mengisi semua tabel, laporan, dan grafik.

## 2. Tech Stack & UI/UX Vibe
- Frontend: React.js (Vite), Tailwind CSS, React Router DOM.
- Visual: Patuhi aturan pada file `DESIGN.md` (Warna Hijau BPJS #008C4A dan Biru BPJS #00539C).

## 3. Fungsionalitas Prototype
- Semua 7 halaman utama harus bisa dinavigasi dengan mulus melalui Sidebar.
- Terapkan interaksi state lokal (UI/UX) dasar: hover state pada tombol, perpindahan Tab, dll.
