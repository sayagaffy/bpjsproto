# Panduan Proyek: Otomasi Laporan Keuangan BPJS

## Tech Stack
- Frontend: React.js (Vite)
- Styling: Tailwind CSS
- Routing: React Router DOM

## Aturan Desain (Stitch MCP)
- Selalu patuhi file `DESIGN.md` yang diekstrak dari Stitch untuk memastikan konsistensi warna (Hijau BPJS #008C4A dan Biru BPJS #00539C) dan tipografi.
- Gunakan skill `react-components` saat mengubah desain HTML/CSS dari Stitch menjadi komponen React yang modular.

## Arsitektur & Kualitas Kode
- Terapkan Functional Programming, hindari side effects.
- Komponen harus dipecah menjadi bagian kecil yang dapat digunakan kembali (reusable).
- Data pada tabel menggunakan data dummy (JSON statis) untuk keperluan prototipe.
